# Scrum poker - mobile App

#### Make your Refinements a little bit more interesting with this simple mobile app for android and iOS! 

### Requirements:
- node `10+` (best `10.15.+`)
- npm `6+` (best `6.4.+`)

### Project setup:
> npm install

### Start expo server:
> npm start

### Start expo server for specific platform:
> npm run start:android

> npm run start:ios

### Run unit tests:
> npm test

### Problems? Try this:
- Clear expo cache:
  > expo r -c
